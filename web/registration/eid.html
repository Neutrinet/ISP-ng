Choice of identification method

We are obligated by telco law in Belgium to register data that must allow to correctly identify the end users.

eID identification

Choose this option to register using your eID. You will be given the opportunity to correct any outdated information later on.

You need to use a Java applet to do this. We're so sorry, there is no other way.

If you're unsure what is contained within the applet, the official source code is here.

Password identification

Choose if you want to be able to log into the vpn using your email address and the password you have choosen at the beginning of this registration.

Privacy

We store the following information when you use the eID:

  • Full name (first + middle + last)
  • Address
  • Date and place of birth
  • Chip number (needed when supplying a PKCS11 id)

We do not process and/or store anything else. To check for yourself, please find the source code at GitHub .
