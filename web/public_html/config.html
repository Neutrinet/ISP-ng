Download your OpenVPN configuration file archive

Prerequisites:

  • Supported platform and version (see below)
  • UDP port 1194 must be open on your firewall
  • Some preliminary debugging skills in case something goes wrong
  • Latest eID middleware (only when using the keypair from your eID)
Note that the eID middleware is required on all platforms where PKCS11 auth (eID) is used. We might be able to simply ship the PKCS11 library in the future, but ATM please download and install the eID middleware.

Linux platform

Download OpenVPN config archive

The file you are about to download contains everything you need. Open a terminal, and run

sudo aptitude install openvpn
cd [download location]
unzip neutrinet.zip
sudo openvpn neutrinet.ovpn # root needed for tun adapter

And you're connected!

Supported configurations

Currently, we support:

  • Debian Wheezy (7.5) with OpenVPN 2.3+ (wheezy-backports)
  • Ubuntu "Trusty Tahr" 14.04 LTS

Is your configuration not supported but are you willing to extensively test your setup? Get in touch, and we'll contact you if we are able to start validating your distro/version.

A note about Network Manager

Due to oversight and/or neglect, the network-manager-openvpn does not allow to configure for PKCS11 modules, used when authenticating with your eID. Multiple bugs reports have been posted to solve this by allowing for an additional config file, without any result thus far.
Please use the openvpn command directly to work around this problem.

OS X platform

Download Tunnelblick config

We advise you to use the Tunnelblick OpenVPN GUI on OSX, which has great support for OpenVPN, ships with recent versions of OpenVPN and integrates well with OSX.

Simply download and install Tunnelblick, then open the Tunnelblick config you downloaded, enter your eID PIN and, presto, you're connected!

Supported configurations

Neutrinet supports OSX 10.6 (Snow Leopard) and up when using Tunnelblick.

Windows platforms

Download OpenVPN config archive

Should you wish to continue using Windows, download the official OpenVPN distribution client (use the Windows Installer) . Unzip the config file archive, right click on the .ovpn file (with the OpenVPN logo) and select 'Start OpenVPN connection'. Done!

Supported configurations

Windows 8.1 Professional/Enterprise with OpenVPN version 2.3.4.

Others might work as well, but do so at your own risk.
